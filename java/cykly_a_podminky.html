Cykly a podmínky

Ukázka porovnání dvou čísel

V následující ukázce uživatel zadá postupně dvě čísla. Program zjistí, které z těchto čísel je větší. V programu si můžete prohlédnout použití podmíněného příkazu if. V závorce za tímto příkazem může být zapsána jakákoliv podmínka. Je-li tato podmínka splněna, program vypíše text "Cislo a je vetsi nez b."; není-li tato podmínka splněna, program vypíše text "Cislo a je mensi nebo rovno cislu b."

Při zápisu podmínek se používají operátory:

  • > (vetší než)
  • < (menší než)
  • >= (větší nebo rovno než)
  • <= (menší nebo rovno než)
  • == (rovná se - pozor, rovnítka jsou skutečně dvě!)
  • != (nerovná se)

Pokud potřebujete zapsat více podmínek najednou, použijete logické spojky:

  • && (konjunkce - a zároveň)
  • || (disjunkce - nebo)
  • ! (negace)

Například podmínce (a >= 3) && !(a > 5) vyhovují čísla větší nebo rovna 3, která nejsou větší než 5. Podmínka je tedy splněna třeba pro číslo 4.

import java.util.Scanner;

class Porovnani {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Napis cele cislo a: ");
        int a;
        a = sc.nextInt();
        System.out.print("Napis cele cislo b: ");
        int b;
        b = sc.nextInt();
        if (a > b) {
            System.out.print("Cislo a je vetsi nez b.");
        } else {
            System.out.print("Cislo a je mensi nebo rovno cislu b.");
        }
    }
}

Úkol:

  1. Napište program, který požádá uživatele o zadání jednoho celého čísla. Poté program vypíše, jestli dané číslo:
    • je kladné;
    • leží mezi čísly -10 a 10;
    • neleží mezi čísly -10 a 10.

Ukázka - jak poznat sudé číslo (číslo dělitelné 2)

V následující ukázce uživatel zadá jedno číslo. Program zjistí, zda je toto číslo sudé nebo liché. V programu se používá operátor %. Tento operátor určí zbytek při celočíselném dělení. Například výsledek operace 17 % 5 je číslo 2, protože 17 děleno 5 je 3, ale nás zajímá zbytek, který je 2. Podobně například 23 % 9 je 5. Všimněte si, že například 15 % 5 je 0, protože číslo 15 je dělitelné pěti. Stejným způsobem poznáme, zda je dané číslo dělitelné dvěma.

import java.util.Scanner;

class Sude {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        System.out.print("Napis cele cislo a: ");
        int a;
        a = sc.nextInt();
        if (a % 2 == 0) {
            System.out.print("Cislo a je sude.");
        } else {
            System.out.print("Cislo a je liche.");
        }
    }
}

Úkoly:

  1. Napište program, který požádá uživatele o zadání jednoho celého čísla. Program poté vypíše, jestli dané číslo:
    • je dělitelné pěti;
    • je dělitelné třemi nebo pěti;
    • není dělitelné třemi ani pěti.
  2. Napište program, který požádá uživatele o zadání tří čísel a určí největší z nich.
  3. Napište program, který požádá uživatele o zadání tří čísel a vypíše je v pořadí od nejmenšího k největšímu.

Ukázka cyklu for - výpis všech lichých čísel od 1 do 99

Při programování často potřebujeme některý příkaz nebo skupinu příkazů zopakovat mnohokrát. K tomuto účelu složí cykly. Nejprve si představíme cyklus for, který nám v následující ukázce pomůže vypsat všechna lichá čísla od 1 do 99. Na začátku cyklu se nastaví hodnota řídící proměnné i na 1. Poté se opakovaně provádí příkaz umístěný mezi složenými závorkami (tzv. tělo cyklu). Při každém opakování se výrazem i+= 2 zvýší hodnota proměnné i o 2. Cyklus se provádí tak dlouho, dokud je splněna podmínka i <= 99. Ve chvíli, kdy hodnota proměnné i překročí 99, je cyklus ukončen a běh programu pokračuje příkazem následujícím za závorkou ukončující tělo cyklu - v našem případě příkaz vypisující text "Konec cyklu :-)".

class VypisLichych {

    public static void main(String[] args) {
        int i;
        for (i = 1; i <= 99; i += 2) {
            System.out.println(i);
        }
        System.out.println("Konec cyklu :-)");
    }
}

Ukázka cyklu for - součet čísel od 1 do 100

V těle cyklu nemusíme nutně něco vypisovat, můžeme třeba hodnoty sčítat. Příkaz s += i zvětší hodnotu proměnné s o hodnotu, která je v dané chvíli zrovna uložena v proměnné i (ta se ale průběžně mění).

class Soucet100 {

    public static void main(String[] args) {
        int s = 0;
        int i;

        for (i = 1; i <= 100; i++) {
            s += i;
        }
        System.out.println("Součet je " + s);
    }
}

Výpis znaků z části Unicode tabulky

Znaky jsou v počítači kódovány čísly. Java používá tzv. Unicode kódování. Prohlédněte si část Unicode tabulky. Z tabulky zjistíte, že například znaku "A" odpovídá v desítkové soustavě kód 65. Následující kód vypíše znaky odpovídající kódům 32 až 127. Poznamenejme, že operátor (char) převede číselnou hodnotu proměnné i na na znak s odpovídajícím kódem. Všimněte si také deklarace proměnné i přímo v hlavičce cyklu. Takto deklarovaná proměnná má platnost pouze při provádění cyklu, po skončení cyklu zanikne. 

           
class Znaky {

    public static void main(String[] args) {

        for (int i = 32; i <= 127; i++) {
            System.out.println((char) i);
        }
    }
}

Úkoly:

  1. Vypište všechna čísla od 100 do 1 sestupně.
  2. Vypište všechny násobky čísla 7 menší než 1000 (vzestupně).
  3. Napište program, který vyzve uživatele, aby zadal přirozené číslo. Program potom vypíše všechna lichá čísla, která jsou menší než zadané číslo.
  4. Napište program, který vyzve uživatele, aby zadal dvě celá čísla. Program potom vypíše všechna čísla, která leží mezi těmito dvěma zadanými čísly. (Pozor, program musí fungovat i v případě, že první zadané číslo je větší než druhé.)
  5. Napište program, který vyzve uživatele, aby zadal dva znaky. Program potom vypíše všechny znaky z Unicode tabulky, které leží mezi těmito dvěma zadanými znaky.
  6. Určete součet všech sudých čísel ležících mezi čísly 1000 a 2000.
  7. Napište program, který vypočítá faktoriál zadaného čísla.

Ukázka cyklu while - program sčítá čísla dokud nezadáme 0

Dalším typem cyklu je cyklus while. Narozdíl od cyklu for je součástí hlavičky cyklu while pouze podmínka. Dokud je podmínka splněna, provádí se opakovaně tělo cyklu, tj. příkazy umístěné mezi složenými závorkami hned za podmínkou. V následující ukázce je podmínka splněna v případě, že je hodnota proměnné i různá od nuly. Pokud tedy uživatel zadá číslo různé od nuly, cyklus pokračuje dalším opakováním. Cyklus je ukončen ve chvíli, kdy uživatel zadá nulu. 

import java.util.Scanner;

class Scitacka1 {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);
        int s = 0;
        int i;   
     
        System.out.print("Napis cislo: ");        
        i = sc.nextInt();

        while (i != 0) {
            s = s + i;
            System.out.print("Mezisoucet je: ");
            System.out.println(s);
            System.out.print("Napis dalsi cislo: ");
            i = sc.nextInt();
        }
    }
}

Ukázka cyklu do-while - program sčítá čísla dokud nezadáme 0

Posledním typem cyklu je cyklus do-while. V případě tohoto cyklu je podmínka uvedena až za tělem cyklu, což je hlavní a jediný rozdíl oproti cyklu while. I v případě, že podmínka v cyklu do-while není splněna při zahájení cyklu, je tělo cyklu alespoň jednou provedeno. To může být v některých situacích výhodné (srovnejte následující ukázku s předchozí). 

import java.util.Scanner;

class Pokus {

    public static void main(String[] args) {
        Scanner sc = new Scanner(System.in);

        int s = 0;
        int i;

        do {
            System.out.print("Napis cislo: ");
            i = sc.nextInt();
            s = s + i;
            System.out.print("Mezisoucet je: ");
            System.out.println(s);
        } while (i != 0);
    }
}

Úkoly:

  1. Vypište všechna čísla od 100 do 1 sestupně. Použijte cyklus while.
  2. Vypište všechny násobky čísla 7 menší než 1000 (vzestupně). Použijte cyklus do-while.
  3. Napište program, který vyzve uživatele, aby zadal přirozené číslo. Program potom vypíše všechna lichá čísla, která jsou menší než zadané číslo. Vyzkoušejte variantu s cyklem while i s cyklem do-while.
  4. Napište program, který postupně násobí celá čísla, která zadává uživatel po spuštění programu. Program průběžně vypisuje aktuální výsledek (součin). Program skončí ve chvíli, kdy uživatel zadá číslo 0 (nula). Použijte cyklus while.
  5. Pokuste se upravit program z předchozí úlohy tak, že použijete cyklus do-while.
  6. Napište program, který čte celá čísla (zadávaná uživatelem po spuštění programu) tak dlouho, dokud uživatel nezadá číslo 0 (nula). U každého zadaného čísla program vypíše, jestli je sudé, nebo liché.
  7. Napište program, který čte celá čísla (zadávaná uživatelem po spuštění programu) tak dlouho, dokud uživatel nezadá číslo 0 (nula). Na závěr program vypíše, kolik zadaných čísel bylo z intervalu <10, 20>.
  8. Napište program, který přečte n čísel zadávaných postupně uživatelem (číslo n zadá uživatel jako první). Pro každý vstup čísla program vypíše:
    Zadej 1. cislo:
    Zadej 2. cislo:
    ...
    Ze zadaných čísel určete, kolik jich leží v intervalu <50, 100>.
  9. Napište program, který přečte n čísel zadávaných postupně uživatelem (číslo n zadá uživatel jako první) a určí největší z nich.

Úloha pro zdatnější

Naprogramujte jednoduchou hru. Cílem hry je uhádnout číslo, které si myslí počítač (toto číslo program náhodně vygeneruje z rozsahu 1 až 100). Hráč má možnost opakovaně zadávat čísla (hádat). Po každému zadání čísla mu počítač napovídá stylem "číslo, které si myslím je menší/větší". Hra úspěšně končí v okamžiku, kdy se podaří hráči číslo uhodnout. Program na závěr vypíše počet pokusů, které hráč k uhodnutí čísla potřeboval.
