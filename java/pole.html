Pole

Ukázka práce s polem - vložení pěti čísel do pole a jejich výpis

V následující ukázce uživatel zadá postupně pět čísel. Tato čísla se uloží do pole předem připraveného deklarací int[] p = new int[5]. Toto pole je typu int, proto je do něj možno ukládat pouze celá čísla. Jeho délka je 5, můžete si jej tedy představit jako 5 přihrádek (pozic) pro celá čísla. Jednotlivé pozice se číslují indexy 0, 1, 2, 3, 4. S poli můžete pracovat podobně jako s proměnnými. Můžete například na první pozici (index 0) v poli vložit číslo 64 příkazem p[0] = 64. Nebo můžete vypsat hodnotu uloženou na druhé pozici (index 1) v poli příkazem System.out.print(p[1]). Hlavní výhody polí se ukáží při práci s cykly - index pole může udávat pomocná proměnná, jejíž hodnoty se mění pomocí cyklu. V následující ukázce plní tuto roli proměnná i, jejíž hodnoty se pomocí cyklu postupně mění od 0 do 4. 

import java.util.Scanner;

class PrvniPole {

    public static void main(String[] args) {
        int[] p = new int[5];
        Scanner sc = new Scanner(System.in);

        for (int i = 0; i <= 4; i++) {
            System.out.print("Napis cislo: ");
            p[i] = sc.nextInt();
        }
        System.out.println("V poli jsou cisla: ");
        for (int i = 0; i <= 4; i++) {
            System.out.print(p[i] + ", ");
        }
    }
}

Úkoly:

  1. Upravte kód předchozí ukázky tak, aby bylo možné do pole vložit 8 čísel a poté byl proveden jejich přehledný výpis na obrazovku.
  2. Upravte kód předchozího cvičení tak, aby program vypsal čísla vložená uživatelem do pole v opačném pořadí (pozpátku).
  3. Vytvořte pole pro uložení teplot naměřených za posledních 10 hodin: 
    public class Teploty {

    public static void main(String[] args) {
        int[] teplota = new int[10];
        teplota[0] = 19;                // 0 hodin (půlnoc :-)
        teplota[1] = 17;                // 1 hodina v noci 
        teplota[2] = 16;
        teplota[3] = 15;
        teplota[4] = 15;
        teplota[5] = 14;
        teplota[6] = 13;
        teplota[7] = 15;                // 7 hodin ráno
        teplota[8] = 18;
        teplota[9] = 21;
    }
}
    Postupně upravujte kód této ukázky podle následujích pokynů:
    • Uživatel zadá hodinu (0–9) a program vypíše teplotu naměřenou v tuto hodinu. Komunikace s programem by měla vypadat nějak takto: 
      Zadej hodinu: 5
V 5:00 byla naměřena teplota 14ºC.
    • Pomocí for cyklu vypište všechny naměřené hodnoty do přehledné tabulky: 
      hodina	teplota
-------------------
0	19
1	17
2	16
      atd.
    • Pomocí for cyklu najděte největší hodnotu v poli (nejvyšší naměřenou teplotu). Vypište tuto hodnotu a čas, kdy byla naměřena.
    • Pomocí for cyklu vypočítejte aritmetický průměr naměřených teplot a vypište jej.

Ukázka - pole náhodných čísel

V následující ukázce vytvoříme pole o délce 100 a pomocí generátoru náhodných čísel jej naplníme náhodnými čísly odpovídajícími hodům kostkou. 

import java.util.Random;

public class Kostka {

    public static void main(String[] args) {
        Random hod = new Random();
        int[] vysledek = new int[100];
        for (int i = 0; i <= 99; i++) {
            vysledek[i] = 1 + hod.nextInt(6);
        }
    }
}

Úkoly:

  1. Pomocí for cyklu vypište všechny výsledky uložené v poli (všech 100 hodů kostkou). Výpis může vypadat například takto (samozřejmě s jinými náhodnými čísly): 6, 3, 6, 4, 6, 2, 1, 4, 1, 4, 3, 2, 4, 4, 6, 6, 4, 5, 3, 2, 5, 6, atd.
  2. Pomocí for cyklu projděte celé pole a spočítejte, kolik je v něm uloženo šestek (četnost šestek při 100 hodech kostkou).
  3. Vytvořte další pole cetnosti (celočíselné pole délky 6) a na jednotlivé pozice tohoto pole uložte četnosti jednotlivých výsledků 1, 2, ..., 6 (na první pozici v poli bude uveden počet jedniček, na druhé pozici v poli počet dvojek, atd.). Poté vypište přehlednou statistiku podle následujícího vzoru (samozřejmě s jinými četnostmi): 
    výsledek    1	2   3	4   5	6
četnost     16	23  14	15  21	11

Úloha pro zdatnější

Naprogramujte hru Mastermind, viz Logik na Wikipedii. Cílem hry je uhodnout čtyři čísla, která má počítač uložená v poli (tato číslo program náhodně vygeneruje z rozsahu 1 až 6). Hráč má možnost opakovaně zadávat čísla (hádat). Důležité je, že hráč musí hledaná čísla uvést i ve správném pořadí. Po každém zadání čtyř čísel mu počítač napovídá podle následujích pravidel:

  1. za každé uhodnuté číslo, které hráč umístil na správné místo v pořadí vypíše program jednu hvězdičku (znak "*");
  2. za každé uhodnuté číslo, které hráč umístil na špatné místo v pořadí vypíše program jeden kroužek (znak "o").

Hádající hráč přitom neví, za která čísla byly přiděleny hodnotící znaky. Hra končí, když hádající hráč uhodne kombinaci (vítězný pokus je tedy ohodnocen čtyřmi hvězdičkami).
